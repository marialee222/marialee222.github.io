How to create a personal website using GitHub Pages

Navigate to https://github.com/

Sign up for GitHub:

      1. enter your email
      2. create a password
      3. enter a username
      4. verify your account

In GitHub, create a new repository

      1. add Repository name: [Owner].github.io
      (NOTE: Owner is your GitHub username)
      2. description (optional): My starter web page
      3. set to Public
      4. do not select include all branches
      5. select the bottom link to Create repository. You will be taken to a page where you can view the contents of your new repository.

In the GitHub repository, select the Settings tab at the top far right

  • Select the Pages option on left sidebar under Code and automation. Here, you can see the Branch information.

In the GitHub repository, select the Code tab at the top far left

      1. upload files to generate your website - click "uploading an existing file" and drag and drop your profile picture, index.html, etc. files from your file system and add to your repository.
      2. Commit - add a short commit message explaining what you did: added profile picture and index file, Commit directly to the master branch, and select Commit changes at the bottom

Open your new personal web page by navigating to [Owner].github.io
